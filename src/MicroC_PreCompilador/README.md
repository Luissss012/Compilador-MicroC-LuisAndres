# Compilador MicroC

Proyecto académico para el desarrollo de un compilador básico del lenguaje MicroC.

## Autor
Luis Andrés

## Descripción
Este proyecto implementa las fases básicas de un compilador:
- Análisis léxico
- Análisis sintáctico
- Generación de código intermedio

## Estado
En desarrollo


## Interfaz gráfica inspirada en Arduino IDE.

Funciones:
- Editor de código
- Consola de salida
- Nuevo / Abrir / Guardar
- Simulación de compilación
